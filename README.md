### 环境

如果你安装了`jekyll`，那你只需要在命令行输入`jekyll serve` 就能在本地浏览器中输入`http://127.0.0.1:4000/`预览主题，对主题的修改也能实时展示（需要强刷浏览器）。

### 本地仿真

```http
jekyll s --host 127.0.0.1 --port 8080
```

然后直接在浏览器打开以下网址即可:

```http
http://127.0.0.1:8080/
```

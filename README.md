### 环境

如果还没有搭建环境，可以通过访问我的主页

```http
https://zhao-guang-nan.github.io/
```
搜索**jeyll window环境搭建**文章来安装Jekyll。
如果你安装了`jekyll`，那你只需要在命令行输入

```http
jekyll s --host 127.0.0.1 --port 8080
```

然后直接在浏览器打开以下网址即可:

```http
http://127.0.0.1:8080/
```

最后就可以在浏览器看到你的整个网页了。
